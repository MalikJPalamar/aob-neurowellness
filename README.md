# AOB Neurowellness Stack

> AI-native breathwork and biometric coaching platform for Alchemy of Breath

## What This Is

A neurowellness platform that combines:
- **On-device breath analysis** (Gemma 4 E2B/E4B)
- **Wearable biometric ingestion** (Whoop, Oura, Apple Health via Terra API)
- **AI coaching** (Claude API for personalized breathwork protocols)
- **Facilitator tools** (dashboard for AOB-certified breathwork guides)
- **Group analytics** (real-time coherence for retreats and workshops)

## Biomarker Tiers

| Tier | Metrics | Source |
|------|---------|--------|
| Respiratory | BF, VT, VE/VCO2, FetCO2, coherence ratio, RRV | On-device mic, PNOE, wearables |
| ANS State | HRV, RHR, skin temp, EDA, SpO2 | Whoop, Oura, Apple Watch |
| Cognitive/Mind | Alpha/theta, frontal asymmetry, vocal prosody, RT, sleep arch | Muse, Hume AI, wearables |
| Metabolic | Recovery score, strain, RMR proxy, VO2 max est | Whoop, PNOE, Apple Watch |
